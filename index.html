Jeu bataille navale

Année : 2022

développement d'un jeu

Informations

Le jeu de la bataille navale se joue à deux joueurs. Chacun dispose d'une grille de jeu sur laquelle il place ses bateaux sans que l'adversaire ne puisse les voir. Les bateaux sont représentés par des cases qui peuvent être horizontales ou verticales et qui sont alignées pour former le bateau. Les bateaux ne peuvent pas se toucher et doivent être placés à une distance d'au moins une case. Une fois que les bateaux sont placés, le jeu commence. Les joueurs se relaient pour tenter de couler les bateaux adverses en lançant des attaques sur des cases de la grille. Si l'attaque touche une case occupée par un bateau, le joueur annonce "touché" et l'adversaire doit marquer la case comme touchée. Si toutes les cases d'un bateau sont touchées, le joueur annonce "coulé" et l'adversaire doit marquer toutes les cases du bateau comme touchées. Si tous les bateaux d'un joueur ont été coulés, l'autre joueur remporte la partie.

  • Visual studio code
